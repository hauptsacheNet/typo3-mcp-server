‚ö†Ô∏è Authorization Header Issue Detected

The MCP endpoint cannot receive Authorization headers. This is a common issue that prevents OAuth authentication from working properly.

Common causes and solutions:

  • Apache configuration: Add this to your .htaccess file: 
    RewriteEngine On
RewriteCond %{HTTP:Authorization} ^(.*)
RewriteRule .* - [e=HTTP_AUTHORIZATION:%1]
  • Site-level authentication: If your site uses HTTP Basic Auth, it may conflict with OAuth headers. Consider using IP-based restrictions instead.
  • Proxy/CDN configuration: Some proxies or CDNs strip Authorization headers. Check your infrastructure settings.
