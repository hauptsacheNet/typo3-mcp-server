üîß Generic MCP Client Setup

For MCP clients that support OAuth and remote connections:

  1. Copy the Integration Name and Server URL below
  2. Configure your client with the integration name (optional but recommended)
  3. Use the Server URL for connection
  4. Your client should automatically discover OAuth endpoints
  5. During authentication, you'll be redirected to this TYPO3 backend
  6. After authorization, your client will receive secure access tokens

Note: Your MCP client must support OAuth 2.0 and automatic endpoint discovery.

How it works:
  1. MCP clients discover OAuth endpoints automatically
    • {baseUrl}/.well-known/oauth-authorization-server
    • {baseUrl}/.well-known/oauth-protected-resource
  2. You'll be redirected to this TYPO3 backend for authentication
  3. After authorization, your MCP client gets secure access
  4. All future requests use OAuth tokens automatically
